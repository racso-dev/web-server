## How to run

### For grading

1. `rm -rf conf`
2. `find . -name "*.class" -type f -delete`
3. `find . -name "*.jar" -type f -delete`
4. `cp myConf ./conf`
5. `javac WebServer.java`
6. `java WebServer`
